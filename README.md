# rural-town-decline-in-Japan

![CI](https://github.com/mmth981/rural-town-decline-in-Japan/actions/workflows/ci.yml/badge.svg)

predictive model of rural and home ownership decline in Japan in 20 years

## Run the Streamlit demo

1. Create a Python virtual environment and activate it (recommended):

```bash
python -m venv .venv
source .venv/bin/activate
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run app.py
```

The app will open in your browser. If running in a remote devcontainer, follow the forwarded port URL from Streamlit.
